# abapmerge

The [abapmerge](https://github.com/larshp/abapmerge) tool creates the [abap2UI5-local](https://github.com/abap2UI5/abap2UI5-local) build by merging ABAP includes automatically into a single file, combining all abap2UI5 classes into one local implementation of an HTTP handler.
