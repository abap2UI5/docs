# Local

abap2UI5-local is a special build that combines all artifacts into a single HTTP handler. You only need to create one additional database table, letting you run abap2UI5 independently of any other installation on the system.

For all details, see the repository: [abap2UI5-local](https://github.com/abap2UI5/abap2UI5-local)
