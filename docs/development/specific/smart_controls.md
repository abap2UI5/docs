# Smart Controls

Smart Controls are metadata-driven UI5 controls that build the UI automatically from OData annotations. abap2UI5 supports them on an experimental basis.

For a complete example, see sample `Z2UI5_CL_DEMO_APP_313`.

::: warning
This is an experimental feature and might not work with every Smart Control type or configuration.
:::
