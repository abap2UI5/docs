# Smart Controls

Smart Controls are metadata-driven UI5 controls that automatically render UI based on OData annotations. abap2UI5 provides experimental support for them.

For a working example, check out sample `Z2UI5_CL_DEMO_APP_313`.

::: warning
This is an experimental feature and may not work with every Smart Control type or configuration.
:::
