# Add-ons

abap2UI5 focuses on core capabilities — rendering views, handling events, and exchanging data. This keeps the framework lean and easy to maintain. The real strength comes from the ecosystem: **ready-to-use add-ons and community projects** that extend abap2UI5 for specific needs.

#### Add-ons

Browse the [abap2UI5-addons](https://github.com/abap2UI5-addons) organization to find what you need. Features include:
- Layout variant handling, similar to classic SAP GUI selection screens or ALV reports
- Table maintenance tools
- Integration with third-party libraries like Chart.js for richer charts
- And many more
